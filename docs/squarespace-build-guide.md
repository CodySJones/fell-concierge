# Fell & Co Squarespace Conversion Guide

This guide is for taking an existing Squarespace 7.1 starter site and quietly converting it into the Fell & Co homepage.

The goal is not to fight the template.
The goal is to strip it back, soften it, and replace generic builder defaults with something calmer and more considered.

## First Principle

If a section feels busy, delete something before adding anything.

Fell & Co should feel:

- spacious
- editorial
- restrained
- warm
- quietly premium

It should not feel:

- startup-like
- glossy
- overly branded
- corporate
- cluttered

## What Squarespace Can Do Well Here

Squarespace 7.1 is flexible enough for this if you keep the build simple:

- large text
- quiet image sections
- restrained buttons
- generous spacing
- a light footer
- calm color themes

Do not try to make it perform like a complex custom site.

## Target Homepage Structure

Use these sections in this exact order:

1. Header
2. Hero
3. Positioning
4. Process
5. Design Profiles
6. Sample Box
7. Services Overview
8. Armitage Connection
9. Closing CTA
10. Footer

## Before You Edit Content

Open your homepage in edit mode, then do this first:

1. Delete any announcement bar
2. Remove newsletter popups for now
3. Remove social icons from the header
4. Remove extra buttons from the header
5. Delete any prebuilt gallery, testimonial slider, or feature grid you do not need
6. Delete any loud background image section you know you will not keep

Keep the page almost empty before rebuilding it.

## Site-Wide Style Setup

Squarespace 7.1 uses `Site Styles` for most of this.
Open:

- `Design`
- `Site Styles`

Then set these foundations.

## Color Direction

Use a very restrained palette.

Suggested values:

- Background: `#F2ECE3`
- Secondary background: `#ECE4D8`
- Text: `#1C1713`
- Muted text: `#6E655C`
- Border: `#D8CCBD`
- Button fill: `#1C1713`
- Button text: `#F2ECE3`

Do not use bright white.
Do not use pure black.
Do not use accent colors unless absolutely necessary.

In Squarespace:

1. Open `Site Styles`
2. Open `Colors`
3. Pick the calmest existing palette as a base
4. Edit that palette instead of starting from something high-contrast
5. Apply the same warm light theme to most sections

Use only two section moods:

- main warm plaster background
- a slightly deeper tonal section every few sections if needed

## Font Direction

Use one serif and one restrained sans.

Recommended feel:

- Serif for headings and brand moments
- Sans for navigation, body text, buttons, labels

If available in Squarespace, good directions are:

- Serif: `Cormorant Garamond`, `IvyPresto`-like alternatives, `Canela`-like alternatives, or another calm high-contrast serif
- Sans: `Inter`, `Suisse`-like alternatives, `Helvetica Neue`-like alternatives, `Avenir`-like alternatives, or another restrained neutral sans

If you do not want to overthink this, use:

- Headings: `Cormorant Garamond`
- Body/UI: `Inter`

In Squarespace:

1. Open `Site Styles`
2. Open `Fonts`
3. Set headings to the serif
4. Set paragraphs, buttons, and navigation to the sans

Typography tuning:

- Heading letter spacing: slightly tight or neutral
- Paragraph letter spacing: neutral
- Button letter spacing: slightly increased
- Navigation letter spacing: slightly increased
- Line height: generous, not loose

## Button Styling

Squarespace lets you style buttons globally in `Site Styles > Buttons`.

Set them like this:

- Corner radius: soft, not pillowy
- Shadow: none
- Primary button fill: near-black
- Primary button text: warm off-white
- Secondary button: transparent or tonal, with a very subtle outline
- Button text: all caps or small caps feeling, with spacing

Keep buttons quiet.
Do not make them oversized.

## Header Conversion

Edit the site header.

Target:

- logo on left
- simple nav on right
- one clean CTA only if needed
- no icons
- no busy utility links

Navigation should be:

- Process
- Design Profiles
- Services
- About
- Start Here

If that still feels crowded, use:

- Process
- Services
- About
- Start Here

Header settings to aim for:

- transparent or same color as page background
- no drop shadow
- medium vertical padding
- thin nav text
- no bold nav styling

Use your Fell & Co logo image as the site logo.
If the logo background is visible, let it blend into the same page tone rather than framing it heavily.

## Section-by-Section Build

## 1. Hero

Use a simple text-first section.

Best Squarespace section type:

- blank section with text block + button block
- or a very simple banner section with image removed or muted

Do not use:

- slideshow
- split-screen sales hero
- overlapping collage layouts

Hero content:

Headline:

`The form and function of remodeling has changed.`

Supporting text:

`Curated, design-led kitchen and bath remodeling with a calmer process, clearer decisions, and more beautiful results.`

Primary CTA:

`Begin Your Design Profile`

Secondary CTA:

`How It Works`

Hero layout instructions:

- keep text aligned left
- keep line length comfortable
- let the heading breathe
- if there is an image, use only one quiet image
- if the image makes it busier, remove it entirely

Hero spacing:

- generous top padding
- generous bottom padding
- no visible box around the content

## 2. Positioning

Use a plain text section directly below the hero.

Suggested content:

Heading:

`A remote-first design studio for kitchen and bath remodels.`

Body:

`Fell & Co. combines timeless design with a structured, technology-enabled process that makes remodeling more beautiful, more efficient, and far less chaotic.`

Layout:

- one text block
- no image
- no icon
- no card

Keep it short.

## 3. Process

Use a very simple multi-column or stacked text layout.

Best section types:

- blank section with 3 columns, then duplicate into a second row
- or a list section made from text blocks only

Do not use icons unless Squarespace forces a format and you can keep them tiny and monochrome.

Section heading:

`A calmer sequence, quietly kept in order.`

Intro:

`The process is designed to reduce overwhelm, not add to it.`

Steps:

`Discover`
Begin with a design profile and tell us about your space.

`Curate`
Receive a tailored direction and, when helpful, a sample box.

`Scan`
Capture your room so the design can move forward with more clarity.

`Refine`
Review finishes, layouts, and visual direction.

`Lock In`
Finalize your rendering, selections, and next steps.

`Move Forward`
Proceed into documents, procurement, or bid comparison as needed.

Spacing guidance:

- allow more vertical spacing than you think you need
- avoid boxed step cards if possible
- if the template forces cards, remove shadows and reduce borders

## 4. Design Profiles

This should feel like an editorial preview, not a quiz module.

Section heading:

`Enduring design directions, thoughtfully distilled.`

Body:

`Rather than beginning with endless options, Fell & Co. begins with a curated point of view.`

Profile list:

- Classic Craftsman
- Vintage Modern
- Natural Minimal
- Traditional Contemporary
- Refined Rustic
- Coastal Quiet

Layout:

- simple two-column text list
- or a stacked list with generous line spacing
- or a quiet image plus list

Do not include too much explanation here.

## 5. Sample Box

Use one image and one text block, or a text-only section if imagery is weak.

Heading:

`Abstract decisions become easier when materials are in hand.`

Body:

`The sample box turns design direction into something tangible. It helps clients compare tone, texture, and finish with less second-guessing.`

Optional CTA:

`Learn More`

Only include that CTA if it leads somewhere real.

Image direction:

- stone
- tile
- wood
- brass
- plaster
- neutral flat-lay

Avoid:

- colorful samples
- mood board clutter
- obviously stocky lifestyle imagery

## 6. Services Overview

Do not use a big pricing table.

Use a simple heading and list.

Heading:

`A restrained set of next steps.`

Intro:

`Fell & Co. offers a structured path forward, with services added only when they are useful to the project.`

Service list:

- Color Consult
- Sample Box
- Lock-In Render
- Revisions
- Selections List
- Construction Documents
- Sample Bid

Layout:

- one or two columns
- quiet text only
- no icons

## 7. Armitage Connection

Keep this understated.

Heading:

`An understated path toward building.`

Body:

`For clients seeking a construction path, Fell & Co. can coordinate next steps with Armitage Interiors.`

This should be short enough to read at a glance.

## 8. Closing CTA

This should feel calm, not salesy.

Heading:

`A calmer place to start.`

Body:

`Begin with a design profile, receive a curated direction, and move forward through a remodeling process that feels considered from the outset.`

Button:

`Start Here`

Use just one button.

## 9. Footer

Keep it extremely light.

Include only:

- Fell & Co.
- Redesigning Remodeling.
- one contact email
- copyright

Optional:

- Instagram

Do not use:

- large link columns
- newsletter block
- category-heavy sitemap footer

## What To Remove From the Squarespace Template

Delete or disable any of these if the starter site includes them:

- testimonial carousels
- team grids
- blog previews
- animated image reveals
- multi-card feature sections
- gallery mosaics
- giant footer menus
- stock icons
- section backgrounds with strong color contrast
- multiple CTA banners

## Mobile Adjustments

After desktop looks right, check mobile carefully.

On mobile:

- reduce section count before reducing spacing
- keep the hero heading large but not overwhelming
- keep line lengths short and readable
- stack buttons if side-by-side feels cramped
- remove any section that starts feeling repetitive

The mobile version should still feel spacious.

## Fastest Way To Get This Live

If you want to publish quickly today:

Keep only these homepage sections:

1. Hero
2. Positioning
3. Process
4. Sample Box
5. Closing CTA

That is enough for a calm first version.

## Editing Order

If you want the easiest build sequence, do it in this order:

1. Clean out the template
2. Set fonts
3. Set colors
4. Set button styles
5. Fix the header
6. Build the hero
7. Build the rest of the sections
8. Simplify the footer
9. Check mobile
10. Publish
