This version is intentionally restrained so you can publish quickly on Squarespace and refine later.

## Goal

A calm, elegant live homepage that feels clear, credible, and easy on the eyes.

## Squarespace Styling Notes

### Colors

Use a very restrained palette:

- Background: warm off-white / bone
- Text: charcoal
- Accent: one muted gray-green or warm taupe only if needed

Suggested values:

- Background: `#F2EBE1`
- Secondary background: `#ECE4DA`
- Text: `#1D1A17`
- Body text: `#676058`
- Border: `#D9CFC3`

### Typography

Use:

- Serif for headings
- Sans serif for body and navigation

In Squarespace, choose the most restrained serif and sans options available.

Aim for:

- elegant serif
- quiet sans
- no aggressive bolding
- generous line height

### Spacing

Prioritize:

- generous top and bottom padding
- wide margins
- one image per section at most
- no dense galleries

### Buttons

Keep buttons simple:

- primary: dark fill, light text
- secondary: outline or quiet tonal fill

Rounded is fine, but keep it subtle.

## Squarespace Build Notes

### Best section types to use

- text block
- image block
- spacer only when necessary
- simple gallery only if very restrained

### Avoid

- sliders
- auto-rotating banners
- parallax
- floating elements
- stacked promo cards
- too many newsletter or pop-up prompts
